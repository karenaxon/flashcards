Flashcards for Learning JS & jQuery

Click on a term or concept to see the definition:

JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

Arithmetic operators perform arithmetic on numbers (literals or variables). These include: + - * /

Variables

Variables are containers for storing data (values).

Naming

Use camelCase for identifier names of variables and functions.

Functions

A function is a block of code designed to perform a particular task.

Methods

Methods are actions that can be performed on objects.

Parameters

Parameters are the names listed in the function definition.

Arguments

Arguments are the real values passed to (and received by) the function.

Return

The return statement stops the execution of a function and returns a value from that function.

Chaining

It is a mechanism of calling a method on another method of the same object.

Strings

Characters inside single or double quotes.

Booleans

A boolean represents one of two values: true or false.

Undefined

The undefined property indicates that a variable has not been assigned a value or not declared at all.

NAN

Not-a-Number value

Alert

The alert() method displays an alert box with a specified message and an OK button.

Concatenation

The concat() method joins two or more strings.
